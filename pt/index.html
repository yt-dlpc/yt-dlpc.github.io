Como instalar yt-dlp no Windows em 4 passos

Sem direitos de administrador. Funciona em qualquer PC com Windows 11 ou Windows 10.

Passo 01

Baixar yt-dlp.exe

Salve em qualquer pasta, ex.: C:\Tools\yt-dlp\ — sem instalação, totalmente portátil.

Passo 02

Abrir PowerShell nessa pasta

Clique na barra de endereço do Explorer, digite powershell, pressione Enter.

Passo 03

Verificar a instalação

Execute yt-dlp --version. Se aparecer um número de versão, está pronto.

Passo 04

Baixar seu primeiro vídeo

Cole uma URL do YouTube e pressione Enter. yt-dlp escolhe automaticamente a melhor qualidade.

Comandos yt-dlp para Windows

Comandos yt-dlp mais úteis — copiar e colar

Execute diretamente no PowerShell ou CMD. Substitua a URL pelo link do seu vídeo ou playlist.

Melhor qualidade de vídeo (mesclagem automática com FFmpeg)
yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"
Converter YouTube para MP3 — melhor qualidade
yt-dlp -x --audio-format mp3 --audio-quality 0 "URL"
Baixar playlist inteira do YouTube
yt-dlp --yes-playlist "PLAYLIST_URL"
Baixar em 4K com legendas em português
yt-dlp -f "bestvideo[height<=2160]+bestaudio" --write-sub --sub-langs pt "URL"

FAQ

Perguntas frequentes sobre yt-dlp para Windows

yt-dlp é seguro para usar no Windows?
Sim. yt-dlp é totalmente open source e hospedado no GitHub. Cada linha de código é auditada publicamente. Sem software adicional, sem telemetria, sem serviços em segundo plano.
yt-dlp funciona no Windows 11?
Sim. yt-dlp.exe funciona no Windows 11 e Windows 10 (64-bit) sem instalação. Baixe o .exe, abra o PowerShell e execute seu primeiro comando.
Preciso de uma GUI para yt-dlp no Windows?
Não. yt-dlp funciona muito bem com comandos simples de copiar e colar no PowerShell. Se preferir interface visual, veja o melhor guia de GUI yt-dlp para Windows 2026.
É legal usar yt-dlp?
O yt-dlp em si é legal — é uma ferramenta open source. Se baixar conteúdo específico é legal depende dos direitos autorais e dos Termos de Serviço da plataforma. Veja o guia completo de segurança e legalidade do yt-dlp.
Como atualizo o yt-dlp no Windows?
Abra o PowerShell na pasta onde yt-dlp.exe está salvo e execute yt-dlp -U. Ele se substitui automaticamente pela versão mais recente.
Preciso de FFmpeg com yt-dlp?
FFmpeg é opcional mas altamente recomendado. O YouTube transmite 4K/8K como faixas de vídeo e áudio separadas — o FFmpeg as mescla. Também é necessário para extração de MP3. Veja o guia de configuração do FFmpeg para Windows.
