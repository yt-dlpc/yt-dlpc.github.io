Guia de configuração

Como configurar uma GUI yt-dlp no Windows

A maioria das GUIs segue o mesmo fluxo de configuração. Em menos de 5 minutos.

Passo 01

Baixar yt-dlp.exe

Obtenha o último yt-dlp.exe na nossa página de download. Salve em C:\Tools\yt-dlp\

Passo 02

Baixar a GUI

Escolha uma GUI da lista acima. Baixe o release da página GitHub do projeto — procure um .exe ou .zip na seção Releases.

Passo 03

Apontar GUI para yt-dlp.exe

A maioria das GUIs pede o caminho para yt-dlp.exe no primeiro uso. Navegue até a pasta C:\Tools\yt-dlp\ e selecione o arquivo.

Passo 04

Adicionar FFmpeg

Baixe ffmpeg.exe e coloque ao lado de yt-dlp.exe. Necessário para 4K/8K e MP3. Guia →

Passo 05

Definir pasta de saída

Nas configurações da GUI, escolha onde os downloads serão salvos. Uma pasta dedicada como C:\Videos\ mantém tudo organizado.

Passo 06

Colar URL e baixar

Cole uma URL do YouTube na GUI, escolha formato e qualidade, e clique em Baixar. Pronto.

Configurações de formato

Configurações de qualidade recomendadas para GUI yt-dlp

Se sua GUI permite definir strings de formato, use estas.

Melhor qualidade de vídeo — mesclagem automática com FFmpeg (recomendado)
bestvideo+bestaudio/best
Limitar a 1080p — bom equilíbrio entre qualidade e tamanho
bestvideo[height<=1080]+bestaudio/best[height<=1080]
Somente MP4 — máxima compatibilidade
bestvideo[ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]
Somente áudio — MP3
-x --audio-format mp3 --audio-quality 0

FAQ

GUI yt-dlp para Windows — perguntas frequentes

Existe uma GUI oficial para yt-dlp?
Não. yt-dlp é uma ferramenta de linha de comando e os desenvolvedores não lançaram uma GUI oficial. Todos os frontends listados aqui são projetos da comunidade.
Ainda preciso instalar yt-dlp ao usar uma GUI?
Depende da GUI. Algumas incluem yt-dlp (como o Parabolic), enquanto outras pedem que você aponte para seu próprio yt-dlp.exe. Verifique a documentação da GUI — se incluir yt-dlp, certifique-se que está atualizado.
Preciso de FFmpeg ao usar uma GUI yt-dlp?
Sim, para downloads 4K/8K e conversão para MP3. Algumas GUIs incluem FFmpeg; outras precisam que você configure o caminho. Coloque ffmpeg.exe na mesma pasta que yt-dlp.exe e a maioria das GUIs encontrará automaticamente. Veja o guia de configuração do FFmpeg.
A GUI diz "yt-dlp não encontrado" — o que fazer?
Vá às Configurações da GUI e defina o caminho do yt-dlp manualmente. Navegue até a pasta onde salvou yt-dlp.exe (ex.: C:\Tools\yt-dlp\yt-dlp.exe) e selecione o arquivo.
GUIs yt-dlp de terceiros são seguras?
As de código aberto vinculadas de suas páginas oficiais do GitHub são geralmente seguras. Sempre baixe da página de Releases do GitHub do projeto — nunca de sites de download de terceiros. Verifique a contagem de estrelas e atividade recente de commits como sinal básico de qualidade.
Prefere a linha de comando afinal?
Entendido — o CLI dá controle total. Veja o cheatsheet de comandos yt-dlp para comandos de copiar e colar cobrindo todos os casos de uso comuns.
