FFmpeg para Windows — Configuração para yt-dlp

Ative mesclas de vídeo+áudio em 4K/8K e conversões para MP3 adicionando o ffmpeg.exe ao seu Windows. O jeito mais rápido e simples é colocar o ffmpeg.exe ao lado do yt-dlp.exe.

⬇️ Obter FFmpeg para Windows Obter yt-dlp.exe

Observação: Os builds para Windows são fornecidos por mantenedores confiáveis listados no site oficial do FFmpeg. Escolha um build recente e extraia localmente.
• Funciona no Windows 11/10

Configuração rápida (recomendado)

  1. Baixe um build para Windows na página de downloads oficiais do FFmpeg.
  2. Extraia o arquivo e localize o ffmpeg.exe (geralmente na pasta bin).
  3. Copie o ffmpeg.exe para a mesma pasta do yt-dlp.exe.
  4. Teste com um vídeo 4K; você deve ver [ffmpeg] Merging formats na saída.

Verificar se o FFmpeg foi detectado

Execute um download rápido no PowerShell:

yt-dlp -f "bestvideo+bestaudio/best" "https://www.youtube.com/watch?v=VIDEO_ID"

Procure por [ffmpeg] Merging formats. Alternativamente, confira via:

ffmpeg -version

(Opcional) Adicionar o FFmpeg ao PATH

  • GUI: Variáveis de Ambiente → Variáveis do Usuário → Path → Novo → cole a pasta bin do FFmpeg → OK.
  • PowerShell (usuário atual): setx PATH "%PATH%;C:\\Path\\to\\ffmpeg\\bin" (reinicie o terminal).

O PATH é opcional para o yt-dlp se o ffmpeg.exe estiver ao lado do executável.

Boas práticas

  • Mantenha o FFmpeg atualizado para o build estável mais recente (correções de codec e desempenho).
  • Coloque o ffprobe.exe junto do ffmpeg.exe para detecção de formatos avançada (opcional).
  • Guarde o yt-dlp.exe e o ffmpeg.exe em uma pasta dedicada como C:\\Tools\\yt-dlp.

Armadilhas comuns

  • “ffmpeg not found”: garanta que o ffmpeg.exe esteja na mesma pasta do yt-dlp.exe ou adicionado ao PATH.
  • Erros de mesclagem: atualize o FFmpeg e o yt-dlp; verifique se o antivírus não está bloqueando; baixe novamente se houver corrupção.
  • Sem capa/metadados no MP3: inclua --embed-thumbnail --embed-metadata e garanta que o FFmpeg esteja presente.

FAQ

O FFmpeg é necessário para todo download?

Não. Ele é necessário para mesclar vídeo/áudio separados (comum em 4K/8K) e para converter áudio (ex.: MP3). Para downloads de stream único, o yt-dlp pode funcionar sem ele.

Onde devo colocar o ffprobe.exe?

Na mesma pasta do ffmpeg.exe é o ideal. O yt-dlp pode usar o ffprobe para melhor detecção de formatos, mas é opcional.

Posso usar PATH em vez de copiar ao lado do yt-dlp.exe?

Sim. Adicione a pasta bin do FFmpeg ao PATH. O yt-dlp o encontrará em todo o sistema.

Legalidade e segurança

Baixe o FFmpeg a partir de fontes listadas no site oficial. Baixe e processe apenas conteúdo ao qual você tem direito. Apontamos para o lançamento oficial do yt-dlp.
