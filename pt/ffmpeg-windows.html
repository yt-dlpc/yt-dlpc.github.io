Guia de configuração

Como instalar FFmpeg para yt-dlp no Windows

Quatro passos, leva menos de 4 minutos.

Passo 01

Baixar FFmpeg para Windows

Vá a ffmpeg.org/download.html e escolha um build Windows. Fontes recomendadas: gyan.dev ou BtbN — ambas listadas na página oficial.

Passo 02

Extrair o arquivo

Extraia o arquivo .zip ou .7z. Abra a pasta extraída e navegue até a subpasta bin — você encontrará ffmpeg.exe lá.

Passo 03

Colocar ao lado de yt-dlp.exe

Copie ffmpeg.exe para a mesma pasta que yt-dlp.exe (ex.: C:\Tools\yt-dlp\). yt-dlp detecta automaticamente — sem configuração necessária.

Passo 04

Verificar se funciona

Baixe qualquer vídeo com yt-dlp. Você deve ver [ffmpeg] Merging formats na saída. Ou execute ffmpeg -version no PowerShell.

Opcional: adicionar FFmpeg ao PATH

Não necessário se ffmpeg.exe estiver ao lado de yt-dlp.exe. Mas útil se quiser FFmpeg disponível em qualquer lugar.

  1. Pesquise Variáveis de Ambiente no Iniciar
  2. Abra Path em Variáveis do usuário
  3. Clique em Novo → cole o caminho para ffmpeg\bin
  4. Clique OK, reinicie o terminal

Solução de problemas

FFmpeg não encontrado ou erros de mesclagem — como corrigir

Erro "ffmpeg not found" no yt-dlp
Verifique se ffmpeg.exe está exatamente na mesma pasta que yt-dlp.exe. Se adicionou FFmpeg ao PATH, feche todas as janelas do terminal e reabra o PowerShell — alterações no PATH só têm efeito em novas sessões. Execute ffmpeg -version para confirmar que está acessível.
Mesclagem falhou ou saída não tem áudio
Atualize tanto yt-dlp (yt-dlp -U) quanto FFmpeg para a versão mais recente. Builds desatualizados do FFmpeg às vezes falham ao mesclar formatos de codec mais novos. Verifique também se o antivírus não está bloqueando ffmpeg.exe.
Antivírus sinaliza ffmpeg.exe como ameaça
Falso positivo. FFmpeg é uma ferramenta open source amplamente usada. Baixe apenas de gyan.dev ou BtbN (linkados da página oficial ffmpeg.org). Adicione uma exceção para o arquivo nas configurações do antivírus se necessário.
Também preciso de ffprobe.exe?
ffprobe é opcional. yt-dlp pode usá-lo para detecção de formato mais precisa, mas não é necessário para mesclagens ou conversões básicas. Se você baixar o build completo do FFmpeg, ffprobe.exe está incluído — apenas coloque ao lado de ffmpeg.exe.
MP3 não tem artwork mesmo com FFmpeg instalado
Certifique-se de incluir --embed-thumbnail --embed-metadata no comando yt-dlp. FFmpeg deve estar presente para essas flags funcionarem. Veja o guia MP3 para o comando completo.
