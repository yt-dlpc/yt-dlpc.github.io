Início rápido após o download

  1. Coloque o yt-dlp.exe em uma pasta como C:\\Users\\VOCÊ\\Downloads ou C:\\Tools\\yt-dlp.
  2. (Recomendado) Coloque o ffmpeg.exe ao lado para mesclas 4K/8K e conversão de áudio (Configuração do FFmpeg).
  3. Abra o PowerShell nessa pasta e execute yt-dlp --help para verificar se está funcionando.
  4. Mantenha atualizado com yt-dlp -U.

Perguntas comuns

Preciso de direitos de administrador para executar o yt-dlp.exe?

Não. O yt-dlp.exe é portátil e roda sem instalação. Admin só é necessário se você optar por adicionar sua pasta ao PATH do sistema.

Como converto para MP3?

Use yt-dlp -x --audio-format mp3 --audio-quality 0 "VIDEO_OR_PLAYLIST_URL". O FFmpeg é obrigatório. Veja o guia YouTube para MP3.

Como baixo uma playlist inteira?

Execute yt-dlp --yes-playlist "PLAYLIST_URL". Veja o guia de playlists para filtros e opções de retomada.

Como mantenho o yt-dlp atualizado?

Execute yt-dlp -U na pasta do executável. Isso substitui pela versão mais recente.
