Após baixar o yt-dlp.exe

Passo 01

Salvar em uma pasta

Coloque yt-dlp.exe em uma pasta permanente como C:\Tools\yt-dlp\ — evite Downloads, que pode ser limpa automaticamente.

Passo 02

Se o navegador alertar

Clique em Manter → Manter mesmo assim no Edge/Chrome. Se aparecer SmartScreen: Mais informações → Executar mesmo assim.

Coloque ffmpeg.exe na mesma pasta. Necessário para mesclar 4K/8K e MP3. Guia →

FAQ

Perguntas frequentes sobre o download

Preciso de direitos de administrador para executar yt-dlp.exe?
Não. yt-dlp.exe é portátil — sem instalação necessária. Administrador só é necessário se quiser adicionar a pasta ao PATH do sistema para todos os usuários.
Como converter para MP3?
Execute yt-dlp -x --audio-format mp3 --audio-quality 0 "URL" no PowerShell. FFmpeg deve estar na mesma pasta. Veja o guia YouTube para MP3.
Como baixar uma playlist inteira?
Execute yt-dlp --yes-playlist "PLAYLIST_URL". Veja o guia de playlists para opções de ordenação, intervalos e retomada.
Como manter o yt-dlp atualizado?
Abra o PowerShell na pasta com yt-dlp.exe e execute yt-dlp -U. Ele se substitui automaticamente pela versão mais recente.
