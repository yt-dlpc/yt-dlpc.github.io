Baixar YouTube em 4K e 8K com yt-dlp no Windows

yt-dlp baixa os melhores streams disponíveis de vídeo e áudio separadamente e os mescla com FFmpeg. Aqui estão os comandos certos para cada cenário.

⬇ Baixar yt-dlp.exe Configurar FFmpeg →

FFmpeg é necessário para mesclagens 4K/8K

Por que vídeo e áudio são separados: YouTube entrega 4K e 8K como streams separados de apenas vídeo e apenas áudio para reduzir largura de banda. yt-dlp baixa ambos e usa FFmpeg para mesclá-los em um único arquivo. Sem FFmpeg, você receberá apenas vídeo — sem áudio.

Comandos

Comandos yt-dlp para download 4K e 8K no Windows

Todos os comandos requerem FFmpeg na mesma pasta que yt-dlp.exe.

Melhor qualidade disponível — mesclagem automática (recomendado)
yt-dlp -f "bestvideo+bestaudio/best" "URL"
4K explícito (2160p) — melhor codec disponível
yt-dlp -f "bestvideo[height>=2160]+bestaudio/best" "URL"
Limitar a 1080p — bom equilíbrio entre qualidade e tamanho
yt-dlp -f "bestvideo[height<=1080]+bestaudio/best" "URL"
Preferir H.264/MP4 — máxima compatibilidade com players e editores
yt-dlp -f "bestvideo[ext=mp4][vcodec*=h264]+bestaudio[ext=m4a]/best[ext=mp4]" "URL"
Preferir AV1 → VP9 → fallback (melhor compressão)
yt-dlp -f "bestvideo[vcodec^=av01]/bestvideo[vcodec^=vp9]/bestvideo+bestaudio/best" "URL"
8K (4320p) se disponível
yt-dlp -f "bestvideo[height>=4320]+bestaudio/best" "URL"

Guia de codecs

AV1 vs VP9 vs H.264 — qual escolher?

YouTube oferece múltiplos codecs para 4K. Veja o que cada um significa para você.

AV1

  • Melhor compressão em 4K/8K
  • Menor tamanho de arquivo para a mesma qualidade
  • Requer hardware moderno para decodificação suave
  • Melhor escolha se seu PC/player suportar

VP9

  • Boa qualidade em 4K, amplamente suportado
  • Arquivos maiores que AV1
  • Funciona na maioria dos players modernos
  • Boa opção padrão para maioria dos usuários

H.264 (MP4)

  • Melhor compatibilidade — funciona em todo lugar
  • Maior tamanho de arquivo em 4K
  • Pode não estar disponível em 4K em todos os vídeos
  • Melhor para edição ou dispositivos mais antigos

FAQ

Download 4K/8K com yt-dlp — perguntas frequentes

O download 4K não tem áudio — por quê?
FFmpeg está faltando ou não está na mesma pasta que yt-dlp.exe. Sem FFmpeg, yt-dlp não consegue mesclar os streams separados de vídeo e áudio. Coloque ffmpeg.exe ao lado de yt-dlp.exe e re-execute o comando. Veja o guia de configuração do FFmpeg.
O vídeo não tem opção 4K — por quê?
Nem todo vídeo do YouTube é enviado em 4K. yt-dlp não faz upscaling — baixa apenas o que a fonte oferece. Use yt-dlp -F "URL" para listar todos os formatos disponíveis para um vídeo específico.
Como ver todos os formatos disponíveis para um vídeo?
Execute yt-dlp -F "URL" (F maiúsculo). Isso lista cada formato disponível com ID, resolução, codec, bitrate e tamanho de arquivo. Você pode então usar um ID de formato específico com -f FORMAT_ID+AUDIO_ID.
Meu player não reproduz arquivos AV1 ou WebM
Use o comando H.264/MP4: -f "bestvideo[ext=mp4][vcodec*=h264]+bestaudio[ext=m4a]/best[ext=mp4]". Note que H.264 em 4K pode não estar disponível para todos os vídeos — os streams 4K do YouTube são frequentemente apenas AV1 ou VP9.
Como verificar se um arquivo baixado é realmente 4K?
Clique com botão direito no arquivo de vídeo no Explorer → Propriedades → aba Detalhes. Você verá a largura e altura do quadro. 4K é 3840×2160. Alternativamente, abra no VLC: Ferramentas → Informações de Mídia → Codec.
