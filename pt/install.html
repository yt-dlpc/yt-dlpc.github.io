Instalar o yt-dlp no Windows

Um guia rápido para configurar o yt-dlp.exe no Windows 11/10, adicioná-lo ao PATH, verificar, atualizar e habilitar o FFmpeg para 4K/8K e MP3.

⬇️ Download do yt-dlp.exe

Dica: Você pode executar o yt-dlp.exe sem instalar nada. Adicioná-lo ao PATH apenas o torna disponível de qualquer pasta.
• Funciona no Windows 11/10

Passo a passo

  1. Baixe o yt-dlp.exe do lançamento oficial no GitHub: versão mais recente.
  2. Mova-o para uma pasta como C:\\Tools\\yt-dlp ou C:\\Users\\VOCÊ\\Downloads.
  3. (Opcional) Adicione a pasta ao seu PATH:
    • Método GUI: Iniciar → pesquise "Variáveis de Ambiente" → Editar variáveis de ambiente → em Variáveis do usuário abra Path → Novo → cole sua pasta (ex.: C:\\Tools\\yt-dlp) → OK.
    • PowerShell (usuário atual): setx PATH "%PATH%;C:\\Tools\\yt-dlp" (reinicie o terminal depois).
  4. Abra o PowerShell nessa pasta (Explorer → Barra de endereços → digite powershell → Enter) e execute:
    yt-dlp --version
  5. Mantenha atualizado:
    yt-dlp -U

FFmpeg (recomendado)

O FFmpeg permite mesclar streams separados de vídeo/áudio (4K/8K) e conversões de áudio (ex.: MP3).

  1. Baixe o ffmpeg.exe de fontes oficiais.
  2. Coloque o ffmpeg.exe ao lado do yt-dlp.exe na mesma pasta.
  3. Verifique baixando um vídeo em alta qualidade; o yt-dlp deve detectar o FFmpeg automaticamente e mesclar formatos.

Detalhes completos: Configuração do FFmpeg para Windows.

Verificações pós-instalação

Faça um teste rápido no PowerShell:

yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"

Confira a versão e o status de atualização:

yt-dlp --version yt-dlp -U

Executar de qualquer pasta

Quando a pasta estiver no PATH, você pode executar o yt-dlp de qualquer lugar. Caso contrário, execute da própria pasta ou informe o caminho completo:

C:\\Tools\\yt-dlp\\yt-dlp.exe --version

Solução de problemas

  • Alerta do SmartScreen? Clique em Mais informações → Executar mesmo assim. É comum em ferramentas portáteis de código aberto.
  • "Comando não encontrado" após atualizar o PATH? Feche e reabra o terminal (o PATH é carregado no início da sessão).
  • "Acesso negado" ou bloqueado? Clique com o direito em yt-dlp.exe → Propriedades → marque Desbloquear, se presente → OK.
  • Erros de mesclagem em 4K/8K? Garanta que o ffmpeg.exe esteja na mesma pasta do yt-dlp.exe.

FAQ

Preciso de direitos de administrador para instalar?

Não. O yt-dlp.exe é portátil. Admin só é necessário se você adicionar o PATH para todos os usuários; para seu usuário, use o método GUI ou setx.

Isso funciona no Windows 7/8 ou ARM?

O yt-dlp mira principalmente versões modernas do Windows. Alguns sistemas antigos podem exigir setups baseados em Python. Para ARM64, verifique as notas oficiais do lançamento para compatibilidade.

Como atualizo para a versão mais recente?

Execute yt-dlp -U. Isso atualiza o executável no local.
