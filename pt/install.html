Instalação passo a passo

Como instalar yt-dlp no Windows em 6 passos

Funciona no Windows 11 e Windows 10 (64-bit). Sem assistente de instalação, sem direitos de administrador.

Passo 01

Baixar yt-dlp.exe

Vá à página de download e salve yt-dlp.exe do release oficial do GitHub.

Passo 02

Salvar em uma pasta dedicada

Mova o arquivo para um local permanente, ex.: C:\Tools\yt-dlp\. Evite Downloads — arquivos ali podem ser limpos automaticamente.

Passo 03

Abrir PowerShell nessa pasta

No Explorer, clique na barra de endereço, digite powershell e pressione Enter. O PowerShell abre já na pasta correta.

Passo 04

Verificar se funciona

Execute yt-dlp --version. Deve aparecer um número de versão como 2025.xx.xx. Se sim, está funcionando.

Passo 05

Atualizar para a versão mais recente

Execute yt-dlp -U. Ele baixa e se substitui pelo release mais novo. Faça isso regularmente.

Passo 06

Instalar FFmpeg (recomendado)

Baixe ffmpeg.exe e coloque na mesma pasta. Necessário para mesclar 4K/8K e MP3. Guia completo →

Opcional mas recomendado

Como adicionar yt-dlp ao PATH no Windows

Adicionar yt-dlp ao PATH permite executar yt-dlp de qualquer pasta no PowerShell ou CMD — sem precisar navegar até a pasta do yt-dlp toda vez.

Método pela interface gráfica (recomendado)

  1. Pressione Win e pesquise Variáveis de Ambiente
  2. Clique em Editar as variáveis de ambiente do sistema
  3. Em Variáveis do usuário, selecione Path → Editar
  4. Clique em Novo e cole o caminho da pasta, ex.: C:\Tools\yt-dlp
  5. Clique OK em todos os diálogos, depois reinicie o terminal

Método pelo PowerShell

Execute isso uma vez no PowerShell — define o PATH só para seu usuário, sem precisar de administrador:

setx PATH "%PATH%;C:\Tools\yt-dlp"

Feche e reabra o PowerShell após executar. Depois teste com yt-dlp --version de qualquer diretório.

Observação: Se aparecer "yt-dlp não é reconhecido" após adicionar ao PATH, feche todas as janelas do terminal e reabra — alterações no PATH só têm efeito em novas sessões.

Altamente recomendado

Configurar FFmpeg para yt-dlp no Windows

FFmpeg é uma ferramenta gratuita que o yt-dlp usa para mesclar streams de vídeo e áudio. Sem ele, downloads em 4K/8K e conversão para MP3 não funcionarão.

O que FFmpeg habilita

  • Downloads de vídeo em 4K e 8K (streams separados mesclados)
  • Extração de áudio MP3, AAC, FLAC, Opus
  • Processamento de formato de vídeo HDR
  • Incorporar legendas em arquivos MKV

Configuração rápida

  1. Baixe ffmpeg.exe pelo guia do FFmpeg
  2. Coloque ffmpeg.exe na mesma pasta que yt-dlp.exe
  3. yt-dlp detecta automaticamente — sem configuração necessária

Problemas comuns

Solução de problemas na instalação do yt-dlp no Windows

Aviso do SmartScreen do Windows ao executar yt-dlp.exe
Clique em Mais informações → Executar mesmo assim. O SmartScreen sinaliza novos executáveis por padrão. yt-dlp.exe é o release oficial open source do GitHub sem modificações.
"yt-dlp não é reconhecido" no PowerShell
Duas opções: execute yt-dlp diretamente da sua pasta, ou adicione a pasta ao PATH (veja a seção acima). Após adicionar ao PATH, feche todas as janelas do terminal e reabra.
"Acesso negado" ou arquivo bloqueado
Clique com botão direito em yt-dlp.exe → Propriedades → na parte inferior marque Desbloquear se a caixa aparecer → clique OK. Isso remove a restrição "baixado da internet" do Windows.
Download 4K/8K retorna só vídeo ou só áudio
FFmpeg não está instalado ou não foi encontrado. Coloque ffmpeg.exe na mesma pasta que yt-dlp.exe. yt-dlp o detectará automaticamente e mesclará os streams.
yt-dlp -U diz "already up to date" mas estou em uma versão antiga
Execute o PowerShell como administrador e tente novamente. Ou baixe o último yt-dlp.exe manualmente da página de download e substitua o arquivo antigo.
