YouTube para MP3 com yt-dlp no Windows — Melhor Qualidade

Baixe áudio do YouTube como MP3 com artwork e tags incorporados. Funciona para vídeos individuais, playlists e canais. Requer FFmpeg para conversão.

⬇ Baixar yt-dlp.exe Configurar FFmpeg →

Comando recomendado

yt-dlp YouTube para MP3 — comando de melhor qualidade

Este único comando faz tudo: converte para MP3, incorpora capa e tags ID3.

MP3 de melhor qualidade com artwork e tags (recomendado)
yt-dlp -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --embed-metadata "VIDEO_URL"

O que cada flag faz

  • -x — extrair apenas áudio
  • --audio-format mp3 — converter para MP3
  • --audio-quality 0 — maior qualidade VBR
  • --embed-thumbnail — incorporar capa
  • --embed-metadata — incorporar tags ID3

Requisitos

  • ffmpeg.exe deve estar na mesma pasta que yt-dlp.exe
  • Sem FFmpeg: conversão e incorporação de artwork não funcionarão
  • Veja o guia de configuração do FFmpeg se necessário
Nota sobre qualidade: O YouTube transmite áudio no formato Opus a ~128–160 kbps. Converter para "MP3 de 320 kbps" mira nessa taxa de bits mas não pode adicionar qualidade além da fonte original. Use --audio-quality 0 para a melhor saída possível do que está disponível.

Mais opções

Todos os comandos de download MP3 do yt-dlp

Copie e cole no PowerShell. Substitua a URL pelo link do seu vídeo ou playlist.

Taxa de bits constante — alvo ~192 kbps
yt-dlp -x --audio-format mp3 --audio-quality 192K --embed-thumbnail --embed-metadata "VIDEO_URL"
Sem re-codificação — manter áudio Opus original (melhor qualidade real)
yt-dlp -f ba -x --audio-format best "VIDEO_URL"
Converter playlist inteira do YouTube para MP3
yt-dlp -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --embed-metadata --yes-playlist -o "%(playlist_index)02d-%(title)s.%(ext)s" "PLAYLIST_URL"
Processar múltiplas URLs de um arquivo de texto em lote
yt-dlp -a links.txt -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --embed-metadata
Salvar em pasta personalizada com nome artista/título
yt-dlp -x --audio-format mp3 --audio-quality 0 -o "C:/Música/%(artist,channel)s/%(title)s.%(ext)s" "VIDEO_URL"

Dicas

Dicas de nomes de arquivo, tags e artwork

Templates de saída

  • %(title)s — título do vídeo
  • %(artist)s — artista (se disponível)
  • %(channel)s — nome do canal
  • %(playlist_index)02d — número de faixa com zeros à esquerda
  • %(ext)s — extensão do arquivo (mp3)

Pular já baixados

Adicione isso a qualquer comando para pular arquivos já convertidos — útil para playlists grandes:

--download-archive downloaded.txt

Cria um arquivo de log e pula entradas em execuções posteriores.

FAQ

YouTube para MP3 com yt-dlp — perguntas frequentes

MP3 de 320 kbps do YouTube é realmente alta qualidade?
O arquivo MP3 será codificado na taxa alvo de 320 kbps, mas a qualidade real de áudio depende da fonte. O YouTube transmite áudio Opus a ~128–160 kbps. Re-codificar para MP3 de 320 kbps não pode adicionar detalhes que não estavam no original. Se qualidade de áudio é prioridade, use -f ba -x --audio-format best para manter o stream Opus original sem re-codificar.
Por que meu MP3 não tem capa ou tags?
FFmpeg deve estar na mesma pasta que yt-dlp.exe, e você precisa de --embed-thumbnail --embed-metadata no comando. Sem FFmpeg, yt-dlp não pode gravar artwork ou tags ID3. Veja o guia de configuração do FFmpeg.
Como converter uma playlist inteira do YouTube para MP3?
Passe uma URL de playlist diretamente — yt-dlp processará todos os vídeos. Adicione --yes-playlist se yt-dlp pedir confirmação, e use -o "%(playlist_index)02d-%(title)s.%(ext)s" para nomes de arquivo numerados e ordenados.
Onde os arquivos MP3 são salvos?
Por padrão, na pasta onde você executou o PowerShell. Use -o "C:/Música/%(title)s.%(ext)s" para controlar o caminho de saída. Barras normais funcionam no Windows.
Posso converter múltiplas URLs de uma vez?
Sim. Crie um arquivo de texto simples (ex.: links.txt) com uma URL por linha, depois execute yt-dlp -a links.txt -x --audio-format mp3 --audio-quality 0.
