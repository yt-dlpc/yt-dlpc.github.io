YouTube para MP3 no Windows (yt-dlp)

Converta vídeos do YouTube ou playlists inteiras para MP3 com melhor qualidade, capa incorporada e tags corretas. Requer FFmpeg para conversão e metadados.

⬇️ Baixar yt-dlp.exe

Nota de qualidade: o YouTube frequentemente entrega áudio em Opus. Converter para “MP3 320 kbps” define esse alvo de taxa de bits, mas não pode superar a qualidade real da fonte.
• Funciona no Windows 11/10

MP3 com melhor qualidade (recomendado)

Recodifique para MP3 com a mais alta qualidade VBR e incorpore capa + metadados (requer FFmpeg):

yt-dlp -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --embed-metadata "VIDEO_URL"
  • --audio-quality 0: maior qualidade VBR para MP3 (preset LAME).
  • --embed-thumbnail e --embed-metadata: adicionam capa e tags ID3 quando possível.
  • Use com playlists passando uma URL de playlist (veja abaixo).

Taxa de bits constante (CBR) — exemplo

Defina uma taxa de bits específica (ex.: ~192 kbps):

yt-dlp -x --audio-format mp3 --audio-quality 192K --embed-thumbnail --embed-metadata "VIDEO_URL"

Sem recodificar (manter original)

Mantenha o melhor fluxo de áudio sem converter para MP3 (geralmente Opus):

yt-dlp -f ba -x --audio-format best "VIDEO_URL"

Converter uma playlist inteira para MP3

Forneça diretamente a URL da playlist ou force o processamento de playlist:

yt-dlp -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --embed-metadata --yes-playlist -o "%(playlist_index)02d-%(title)s.%(ext)s" "PLAYLIST_URL"
  • -o usa um template de saída com índices com zero à esquerda para manter a ordem correta.
  • Adicione --download-archive downloaded.txt para pular itens já convertidos.

Nomes de arquivos e pastas

yt-dlp -x --audio-format mp3 -o "C:/Music/%(artist,channel)s/%(title)s.%(ext)s" "VIDEO_URL"

Organize bibliotecas usando artista/canal e título. Em caminhos do Windows use / ou \\ com escape.

Dicas de tags e capa

  • Use --embed-thumbnail --embed-metadata para capa e ID3.
  • Aprimore as tags com --parse-metadata se a sua fonte tiver títulos estruturados.
  • O FFmpeg deve estar presente para a incorporação funcionar.

Perguntas frequentes (FAQ)

MP3 de 320 kbps é sempre melhor?

Nem sempre. O YouTube costuma fornecer Opus por volta de 160 kbps (porém eficiente). Recodificar para "320 kbps" define a meta de taxa de bits do MP3, mas não adiciona detalhes além da fonte.

Como procesar vários links em lote?

Coloque as URLs em um arquivo de texto e use yt-dlp -a links.txt -x --audio-format mp3 --audio-quality 0.

Os arquivos não têm capa?

Confirme que ffmpeg.exe está na mesma pasta que o yt-dlp.exe e inclua --embed-thumbnail --embed-metadata.

Onde os arquivos são salvos?

Por padrão, na pasta atual. Controle isso com -o, por exemplo: -o "C:/Music/%(title)s.%(ext)s".
