Come installare FFmpeg per yt-dlp su Windows

Quattro passaggi, richiede meno di 4 minuti.

Passo 01

Scarica FFmpeg per Windows

Vai su ffmpeg.org/download.html e scegli un build Windows. Fonti consigliate: gyan.dev o BtbN — elencate nella pagina ufficiale.

Passo 02

Estrai l'archivio

Estrai il file .zip o .7z. Apri la cartella estratta e naviga nella sottocartella bin — troverai ffmpeg.exe lì.

Passo 03

Metti accanto a yt-dlp.exe

Copia ffmpeg.exe nella stessa cartella di yt-dlp.exe (es.: C:\Tools\yt-dlp\). yt-dlp lo rileva automaticamente — nessuna configurazione necessaria.

Passo 04

Verifica che funzioni

Scarica qualsiasi video con yt-dlp. Dovresti vedere [ffmpeg] Merging formats nell'output. Oppure esegui ffmpeg -version in PowerShell.

Opzionale: aggiungi FFmpeg al PATH

Non necessario se ffmpeg.exe è accanto a yt-dlp.exe. Ma utile se vuoi FFmpeg disponibile ovunque.

  1. Cerca Variabili d'ambiente nel menu Start
  2. Apri Path nelle Variabili utente
  3. Clicca su Nuovo → incolla il percorso verso ffmpeg\bin
  4. Clicca OK, riavvia il terminale

Risoluzione problemi

FFmpeg non trovato o errori di unione — come risolvere

Errore "ffmpeg not found" in yt-dlp
Controlla che ffmpeg.exe sia esattamente nella stessa cartella di yt-dlp.exe. Se hai aggiunto FFmpeg al PATH, chiudi tutte le finestre del terminale e riapri PowerShell — le modifiche al PATH hanno effetto solo nelle nuove sessioni. Esegui ffmpeg -version per confermare che è accessibile.
Unione fallita o l'output non ha audio
Aggiorna yt-dlp (yt-dlp -U) e FFmpeg all'ultima versione. I build obsoleti di FFmpeg a volte non riescono a unire i nuovi formati di codec. Controlla anche che l'antivirus non stia bloccando ffmpeg.exe.
L'antivirus segnala ffmpeg.exe come minaccia
Falso positivo. FFmpeg è uno strumento open source ampiamente usato. Scarica solo da gyan.dev o BtbN (linkati dalla pagina ufficiale ffmpeg.org). Aggiungi un'eccezione per il file nelle impostazioni dell'antivirus se necessario.
Ho bisogno anche di ffprobe.exe?
ffprobe è opzionale. yt-dlp può usarlo per un rilevamento del formato più preciso, ma non è necessario per le unioni o conversioni di base. Se scarichi il build completo di FFmpeg, ffprobe.exe è incluso — mettilo semplicemente accanto a ffmpeg.exe.
L'MP3 non ha copertina anche con FFmpeg installato
Assicurati di includere --embed-thumbnail --embed-metadata nel comando yt-dlp. FFmpeg deve essere presente perché questi flag funzionino. Consulta la guida MP3 per il comando completo.
