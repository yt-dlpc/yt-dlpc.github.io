Scaricare Playlist & Canali YouTube con yt-dlp su Windows

Scarica playlist intere o elementi specifici, mantieni l'ordine, riprendi i download interrotti e salta i file già salvati. Funziona anche per i canali.

⬇ Scarica yt-dlp.exe

Aggiorna prima di lavori grandi: yt-dlp -U

Comandi

Comandi yt-dlp per scaricare playlist su Windows

Tutti i comandi vengono eseguiti in PowerShell dalla cartella dove è salvato yt-dlp.exe.

Scaricare una playlist intera — migliore qualità
yt-dlp --yes-playlist "PLAYLIST_URL"
Mantenere l'ordine della playlist nei nomi file (consigliato)
yt-dlp --yes-playlist -o "%(playlist_index)02d-%(title)s.%(ext)s" "PLAYLIST_URL"
Scaricare solo elementi specifici (es.: 1, 5, da 10 a 12)
yt-dlp --playlist-items 1,5,10-12 "PLAYLIST_URL"
Scaricare un intervallo (elementi da 5 a 15)
yt-dlp --playlist-start 5 --playlist-end 15 "PLAYLIST_URL"
Riprendere + saltare i file già scaricati
yt-dlp --continue --download-archive downloaded.txt "PLAYLIST_URL"
Scaricare un intero canale YouTube (tutti i video)
yt-dlp "https://www.youtube.com/@NomeCanale/videos" -o "%(upload_date>%Y-%m-%d)s-%(title)s.%(ext)s"

Organizzazione

Cartelle, nomi file e template di output

Usa i template di output per creare automaticamente strutture di cartelle organizzate.

Una cartella per playlist, file numerati
yt-dlp -o "C:/Video/%(playlist)s/%(playlist_index)02d-%(title)s.%(ext)s" "PLAYLIST_URL"
Video del canale ordinati per data di caricamento
yt-dlp -o "C:/Video/%(channel)s/%(upload_date>%Y-%m-%d)s-%(title)s.%(ext)s" "CHANNEL_URL"
Sincronizzazione incrementale del canale — solo nuovi video
yt-dlp --download-archive archive.txt --dateafter today-30days "CHANNEL_URL"

FAQ

Download playlist — domande frequenti

Perché alcuni elementi della playlist vengono saltati?
Il video potrebbe essere privato, bloccato per area geografica, con restrizione d'età o già nel tuo file --download-archive. Controlla l'output del terminale per il motivo specifico. Per contenuti privati o con restrizione d'età, prova --cookies-from-browser chrome.
Come scarico solo in 1080p da una playlist?
Aggiungi un filtro di formato: -f "bestvideo[height<=1080]+bestaudio/best[height<=1080]". Per le unioni 4K/8K, assicurati che FFmpeg sia nella stessa cartella di yt-dlp.exe.
I file sono in disordine dopo il download — come si risolve?
Aggiungi -o "%(playlist_index)02d-%(title)s.%(ext)s" al comando. Il prefisso %(playlist_index)02d aggiunge zeri al numero di traccia in modo che i file vengano ordinati correttamente in qualsiasi file manager.
Come tengo solo i nuovi video alla riesecuzione?
Usa --download-archive archive.txt. yt-dlp scrive l'ID di ogni video scaricato in quel file e salta tutti gli ID già elencati nelle esecuzioni future. Il file viene creato automaticamente se non esiste.
Posso scaricare una playlist privata?
Sì, se hai effettuato l'accesso all'account proprietario. Usa --cookies-from-browser chrome (o edge/firefox) per passare i tuoi cookie di sessione a yt-dlp.
