Installazione passo dopo passo

Come installare yt-dlp su Windows in 6 passaggi

Compatibile con Windows 11 e Windows 10 (64 bit). Senza wizard di installazione, senza permessi di amministratore.

Passo 01

Scarica yt-dlp.exe

Vai alla pagina di download e salva yt-dlp.exe dalla release ufficiale GitHub.

Passo 02

Salva in una cartella dedicata

Sposta il file in una posizione permanente, es.: C:\Tools\yt-dlp\. Evita Download — i file lì possono essere eliminati automaticamente.

Passo 03

Apri PowerShell in quella cartella

In Esplora file, clicca sulla barra degli indirizzi, digita powershell e premi Invio. PowerShell si apre direttamente nella cartella giusta.

Passo 04

Verifica che funzioni

Esegui yt-dlp --version. Dovrebbe apparire un numero di versione come 2025.xx.xx. Se sì, funziona.

Passo 05

Aggiorna all'ultima versione

Esegui yt-dlp -U. Scarica e si sostituisce con la release più recente. Fallo regolarmente.

Passo 06

Installa FFmpeg (consigliato)

Scarica ffmpeg.exe e mettilo nella stessa cartella. Necessario per unire 4K/8K e MP3. Guida completa →

Opzionale ma consigliato

Come aggiungere yt-dlp al PATH su Windows

Aggiungere yt-dlp al PATH consente di eseguire yt-dlp da qualsiasi cartella in PowerShell o CMD — senza dover navigare alla cartella yt-dlp ogni volta.

Metodo grafico (consigliato)

  1. Premi Win e cerca Variabili d'ambiente
  2. Clicca su Modifica le variabili d'ambiente di sistema
  3. In Variabili utente, seleziona Path → Modifica
  4. Clicca su Nuovo e incolla il percorso della cartella, es.: C:\Tools\yt-dlp
  5. Clicca OK su tutti i dialoghi, poi riavvia il terminale

Metodo PowerShell

Esegui questo una volta in PowerShell — imposta il PATH solo per il tuo utente, senza permessi di amministratore:

setx PATH "%PATH%;C:\Tools\yt-dlp"

Chiudi e riapri PowerShell dopo. Testa con yt-dlp --version da qualsiasi directory.

Nota: Se appare "yt-dlp non è riconosciuto come comando" dopo aver aggiunto al PATH, chiudi tutte le finestre del terminale e riaprile — le modifiche al PATH hanno effetto solo nelle nuove sessioni.

Verifiche post-installazione

Verificare l'installazione e mantenere yt-dlp aggiornato

Verificare la versione installata
yt-dlp --version
Aggiornare all'ultima release
yt-dlp -U
Testare con un vero download (migliore qualità)
yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"
Se yt-dlp non è nel PATH — esegui direttamente dalla cartella
C:\Tools\yt-dlp\yt-dlp.exe --version

Problemi comuni

Risolvere problemi di installazione di yt-dlp su Windows

Avviso SmartScreen di Windows all'esecuzione di yt-dlp.exe
Clicca su Ulteriori informazioni → Esegui comunque. SmartScreen segnala i nuovi eseguibili per impostazione predefinita. yt-dlp.exe è la release ufficiale open source di GitHub senza modifiche.
"yt-dlp non è riconosciuto" in PowerShell
Due opzioni: esegui yt-dlp direttamente dalla sua cartella, oppure aggiungi la cartella al PATH (vedi sezione sopra). Dopo aver aggiunto al PATH, chiudi tutte le finestre del terminale e riaprile.
"Accesso negato" o file bloccato
Clicca con il tasto destro su yt-dlp.exe → Proprietà → in basso spunta Sblocca se appare la casella → clicca OK. Questo rimuove la restrizione di Windows "scaricato da Internet".
Il download 4K/8K restituisce solo video o solo audio
FFmpeg non è installato o non è trovato. Metti ffmpeg.exe nella stessa cartella di yt-dlp.exe. yt-dlp lo rileverà automaticamente e unirà i flussi.
