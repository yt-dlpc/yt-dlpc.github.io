Quick start

How to install yt-dlp on Windows in 4 steps

No admin rights needed. Works on any Windows 11 or Windows 10 machine.

Step 01

Download yt-dlp.exe

Save to any folder, e.g. C:\Tools\yt-dlp\ — no install, fully portable.

Step 02

Open PowerShell in that folder

Click the address bar in Explorer, type powershell, press Enter.

Step 03

Verify the install

Run yt-dlp --version. If you see a version number, you're ready.

Step 04

Download your first video

Paste a YouTube URL and press Enter. yt-dlp picks the best quality automatically.

yt-dlp commands for Windows

Most useful yt-dlp commands — copy and paste

Run these directly in PowerShell or CMD. Replace the URL with your video or playlist link.

Best quality video (auto-merge with FFmpeg)
yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"
Convert YouTube to MP3 — best quality
yt-dlp -x --audio-format mp3 --audio-quality 0 "URL"
Download entire YouTube playlist
yt-dlp --yes-playlist "PLAYLIST_URL"
Download in 4K with English subtitles
yt-dlp -f "bestvideo[height<=2160]+bestaudio" --write-sub --sub-langs en "URL"

FAQ

Frequently asked questions about yt-dlp for Windows

Is yt-dlp safe to use on Windows?
Yes. yt-dlp is fully open source and hosted on GitHub. Every line of code is publicly audited. No bundled software, no telemetry, no background services.
Does yt-dlp work on Windows 11?
Yes. yt-dlp.exe runs on Windows 11 and Windows 10 (64-bit) with no installation. Download the .exe, open PowerShell, and run your first command.
Do I need a GUI for yt-dlp on Windows?
No. yt-dlp works great with simple copy-paste commands in PowerShell. If you prefer a visual interface, see the best yt-dlp GUI guide for Windows 2026.
Is yt-dlp legal to use?
yt-dlp itself is legal — it's an open-source tool. Whether downloading specific content is lawful depends on its copyright and the platform's Terms of Service. See the full yt-dlp safe and legal guide.
How do I update yt-dlp on Windows?
Open PowerShell in the folder where yt-dlp.exe is saved and run yt-dlp -U. It replaces itself with the latest version automatically.
Do I need FFmpeg with yt-dlp?
FFmpeg is optional but strongly recommended. YouTube streams 4K/8K as separate video and audio tracks — FFmpeg merges them. It's also required for MP3 extraction. See the FFmpeg setup guide for Windows.
