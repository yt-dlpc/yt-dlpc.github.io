How it works (2-minute setup)

  1. Download yt-dlp.exe from the official GitHub release.
  2. Place it in a folder like C:\Users\YOU\Downloads or C:\Tools\yt-dlp.
  3. (Recommended) Put ffmpeg.exe next to it for 4K/8K merges and audio conversion.
  4. Open PowerShell in that folder and run yt-dlp -U to stay up to date.

Popular commands (copy & paste)

Best quality video (auto-merge)

yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"

Download entire playlist

yt-dlp --yes-playlist "PLAYLIST_URL"

Convert to MP3 (best quality)

yt-dlp -x --audio-format mp3 --audio-quality 0 "VIDEO_OR_PLAYLIST_URL"

Download with subtitles

yt-dlp --write-auto-sub --sub-format srt --sub-langs "en,*" "URL"

Limit to 1080p

yt-dlp -f "bestvideo[height<=1080]+bestaudio/best" "URL"
