Installation étape par étape

Comment installer yt-dlp sur Windows en 6 étapes

Fonctionne sur Windows 11 et Windows 10 (64 bits). Sans assistant d'installation, sans droits administrateur.

Étape 01

Télécharger yt-dlp.exe

Allez à la page de téléchargement et enregistrez yt-dlp.exe depuis la release officielle GitHub.

Étape 02

Enregistrer dans un dossier dédié

Déplacez le fichier vers un emplacement permanent, ex. : C:\Tools\yt-dlp\. Évitez Téléchargements — les fichiers peuvent y être nettoyés automatiquement.

Étape 03

Ouvrir PowerShell dans ce dossier

Dans l'Explorateur, cliquez sur la barre d'adresse, tapez powershell et appuyez sur Entrée. PowerShell s'ouvre directement dans le bon dossier.

Étape 04

Vérifier que ça fonctionne

Exécutez yt-dlp --version. Un numéro de version comme 2025.xx.xx doit s'afficher. Si oui, c'est fonctionnel.

Étape 05

Mettre à jour vers la dernière version

Exécutez yt-dlp -U. Il télécharge et se remplace par la release la plus récente. Faites-le régulièrement.

Étape 06

Installer FFmpeg (recommandé)

Téléchargez ffmpeg.exe et placez-le dans le même dossier. Nécessaire pour fusionner la 4K/8K et le MP3. Guide complet →

Facultatif mais recommandé

Comment ajouter yt-dlp au PATH sous Windows

Ajouter yt-dlp au PATH permet d'exécuter yt-dlp depuis n'importe quel dossier dans PowerShell ou CMD — sans naviguer vers le dossier yt-dlp à chaque fois.

Méthode graphique (recommandée)

  1. Appuyez sur Win et recherchez Variables d'environnement
  2. Cliquez sur Modifier les variables d'environnement système
  3. Dans Variables utilisateur, sélectionnez Path → Modifier
  4. Cliquez sur Nouveau et collez le chemin du dossier, ex. : C:\Tools\yt-dlp
  5. Cliquez OK sur tous les dialogues, puis redémarrez le terminal

Méthode PowerShell

Exécutez ceci une fois dans PowerShell — définit le PATH uniquement pour votre utilisateur, sans droits administrateur :

setx PATH "%PATH%;C:\Tools\yt-dlp"

Fermez et rouvrez PowerShell après l'exécution. Testez ensuite avec yt-dlp --version depuis n'importe quel répertoire.

Remarque : Si "yt-dlp n'est pas reconnu" apparaît après avoir ajouté au PATH, fermez toutes les fenêtres du terminal et rouvrez-les — les modifications du PATH ne prennent effet que dans les nouvelles sessions.

Vérifications post-installation

Vérifier l'installation et maintenir yt-dlp à jour

Vérifier la version installée
yt-dlp --version
Mettre à jour vers la dernière release
yt-dlp -U
Tester avec un vrai téléchargement (meilleure qualité)
yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"
Si yt-dlp n'est pas dans le PATH — exécutez directement depuis le dossier
C:\Tools\yt-dlp\yt-dlp.exe --version

Problèmes courants

Résoudre les problèmes d'installation de yt-dlp sous Windows

Avertissement SmartScreen de Windows lors de l'exécution de yt-dlp.exe
Cliquez sur Informations complémentaires → Exécuter quand même. SmartScreen signale les nouveaux exécutables par défaut. yt-dlp.exe est la release officielle open source de GitHub sans modification.
"yt-dlp n'est pas reconnu" dans PowerShell
Deux options : exécutez yt-dlp directement depuis son dossier, ou ajoutez le dossier au PATH (voir la section ci-dessus). Après avoir ajouté au PATH, fermez toutes les fenêtres du terminal et rouvrez-les.
"Accès refusé" ou fichier bloqué
Clic droit sur yt-dlp.exe → Propriétés → en bas cochez Débloquer si la case apparaît → cliquez OK. Cela supprime la restriction "téléchargé depuis Internet" de Windows.
Le téléchargement 4K/8K renvoie seulement la vidéo ou seulement l'audio
FFmpeg n'est pas installé ou introuvable. Placez ffmpeg.exe dans le même dossier que yt-dlp.exe. yt-dlp le détectera automatiquement et fusionnera les flux.
