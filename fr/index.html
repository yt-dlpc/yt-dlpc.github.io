Démarrage rapide

Comment installer yt-dlp sur Windows en 4 étapes

Sans droits administrateur. Fonctionne sur tout PC sous Windows 11 ou Windows 10.

Étape 01

Télécharger yt-dlp.exe

Enregistrez dans n'importe quel dossier, ex. : C:\Tools\yt-dlp\ — sans installation, entièrement portable.

Étape 02

Ouvrir PowerShell dans ce dossier

Cliquez sur la barre d'adresse de l'Explorateur, tapez powershell, appuyez sur Entrée.

Étape 03

Vérifier l'installation

Exécutez yt-dlp --version. Si un numéro de version s'affiche, c'est prêt.

Étape 04

Télécharger votre première vidéo

Collez une URL YouTube et appuyez sur Entrée. yt-dlp choisit automatiquement la meilleure qualité.

Commandes yt-dlp pour Windows

Commandes yt-dlp les plus utiles — copier-coller

Exécutez directement dans PowerShell ou CMD. Remplacez l'URL par votre lien vidéo ou playlist.

Meilleure qualité vidéo (fusion automatique avec FFmpeg)
yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"
Convertir YouTube en MP3 — meilleure qualité
yt-dlp -x --audio-format mp3 --audio-quality 0 "URL"
Télécharger une playlist YouTube entière
yt-dlp --yes-playlist "PLAYLIST_URL"
Télécharger en 4K avec sous-titres en français
yt-dlp -f "bestvideo[height<=2160]+bestaudio" --write-sub --sub-langs fr "URL"

FAQ

Questions fréquentes sur yt-dlp pour Windows

yt-dlp est-il sûr à utiliser sous Windows ?
Oui. yt-dlp est entièrement open source et hébergé sur GitHub. Chaque ligne de code est auditée publiquement. Sans logiciel supplémentaire, sans télémétrie, sans services en arrière-plan.
yt-dlp fonctionne-t-il sur Windows 11 ?
Oui. yt-dlp.exe fonctionne sur Windows 11 et Windows 10 (64 bits) sans installation. Téléchargez le .exe, ouvrez PowerShell et exécutez votre première commande.
Ai-je besoin d'une GUI pour yt-dlp sous Windows ?
Non. yt-dlp fonctionne très bien avec des commandes simples à copier-coller dans PowerShell. Si vous préférez une interface visuelle, consultez le guide des meilleures GUI yt-dlp pour Windows 2026.
Est-ce légal d'utiliser yt-dlp ?
yt-dlp lui-même est légal — c'est un outil open source. Si télécharger un contenu spécifique est légal dépend des droits d'auteur et des Conditions Générales d'Utilisation de la plateforme. Consultez le guide complet sur la sécurité et la légalité de yt-dlp.
Comment mettre à jour yt-dlp sous Windows ?
Ouvrez PowerShell dans le dossier où yt-dlp.exe est enregistré et exécutez yt-dlp -U. Il se remplace automatiquement par la dernière version.
Ai-je besoin de FFmpeg avec yt-dlp ?
FFmpeg est facultatif mais très recommandé. YouTube diffuse la 4K/8K en flux vidéo et audio séparés — FFmpeg les fusionne. Il est aussi nécessaire pour l'extraction MP3. Consultez le guide de configuration FFmpeg pour Windows.
