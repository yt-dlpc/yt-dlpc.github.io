Politique de Confidentialité

yt-dlp Windows Helper est un site non officiel maintenu par la communauté. Nous collectons des données minimales et sommes transparents sur ce que nous utilisons et pourquoi.

Qui nous sommes

yt-dlp Windows Helper est un site non officiel maintenu par la communauté qui fournit des guides et des liens directs vers les releases officielles de yt-dlp sur GitHub. Nous ne sommes pas affiliés à YouTube, Google ou aux développeurs officiels de yt-dlp.

Ce que nous collectons (et ce que nous ne collectons pas)

  • Pas de comptes, pas de formulaires : Nous ne proposons pas de comptes utilisateur et ne collectons pas d'informations via des formulaires.
  • Analytics (Google Analytics 4) : Nous utilisons GA4 pour comprendre les tendances d'utilisation agrégées et non identifiantes — vues de pages, clics sur les boutons, téléchargements. GA4 peut collecter des données d'appareil et d'événements telles que les URLs de pages, les référents, la géolocalisation approximative et les attributs techniques. Les adresses IP sont traitées par Google pour la géolocalisation puis supprimées par défaut.
  • Logs d'hébergement (GitHub Pages) : Ce site est hébergé sur GitHub Pages. GitHub peut collecter des logs de serveur web standard (adresse IP, user agent) pour des raisons de sécurité. Consultez la Déclaration de Confidentialité GitHub.
  • LocalStorage : Nous stockons un flag cookieConsent dans le localStorage de votre navigateur lorsque vous acceptez la bannière de cookies.

Cookies & stockage local

  • Cookies Google Analytics (_ga, _ga_<id>) — mesure d'usage anonyme.
  • LocalStorage : cookieConsent — mémorise votre choix de cookies.

Vous pouvez gérer les cookies via les paramètres de votre navigateur, utiliser le module complémentaire de désactivation Google Analytics, ou effacer les données du site dans votre navigateur.

Pourquoi nous traitons les données

  • Consentement : Les cookies analytics sont utilisés avec votre consentement via notre bannière de cookies.
  • Intérêts légitimes : Les analytics agrégés nous aident à maintenir et améliorer le contenu et les performances du site.

Partage des données & sous-traitants

  • Google (Analytics) : Traite les données analytics en notre nom. Consultez la Politique de Confidentialité Google.
  • GitHub (Pages) : Héberge ce site. Consultez la Confidentialité GitHub.

Nous ne vendons pas vos informations personnelles.

Conservation

  • Données d'événements GA4 : Conservées par Google jusqu'à 14 mois (paramètre par défaut de la propriété).
  • LocalStorage : Le flag cookieConsent persiste jusqu'à ce que vous effaciez les données du site.
  • Logs d'hébergement : Contrôlés par GitHub — consultez leur politique.

Transferts internationaux

Les prestataires d'analytics et d'hébergement peuvent traiter des données dans plusieurs juridictions, dont les États-Unis. Nous nous appuyons sur leurs mesures de protection et politiques documentées.

Vos droits & choix

  • Utilisez le module de désactivation GA pour empêcher la collecte analytics.
  • Bloquez ou supprimez les cookies via les paramètres du navigateur.
  • Do Not Track (DNT) : nous ne répondons pas actuellement aux signaux DNT.
  • Selon votre juridiction (UE/EEE, Californie, etc.) vous pouvez avoir des droits d'accès, de rectification ou de suppression des données. Contactez-nous pour faire une demande.

Confidentialité des enfants

Ce site ne s'adresse pas aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations personnelles auprès d'enfants.

Contact

Pour les questions de confidentialité, contactez-nous via le dépôt GitHub du projet ou le lien de contact du site.

Modifications de cette politique

Nous pouvons mettre à jour cette Politique de Confidentialité de temps en temps. La date de « Dernière mise à jour » en haut reflète les dernières modifications.
