Comment installer FFmpeg pour yt-dlp sous Windows

Quatre étapes, prend moins de 4 minutes.

Étape 01

Télécharger FFmpeg pour Windows

Allez sur ffmpeg.org/download.html et choisissez un build Windows. Sources recommandées : gyan.dev ou BtbN — listées sur la page officielle.

Étape 02

Extraire l'archive

Extrayez le fichier .zip ou .7z. Ouvrez le dossier extrait et naviguez jusqu'au sous-dossier bin — vous y trouverez ffmpeg.exe.

Étape 03

Placer à côté de yt-dlp.exe

Copiez ffmpeg.exe dans le même dossier que yt-dlp.exe (ex. : C:\Tools\yt-dlp\). yt-dlp le détecte automatiquement — aucune configuration nécessaire.

Étape 04

Vérifier que ça fonctionne

Téléchargez n'importe quelle vidéo avec yt-dlp. Vous devriez voir [ffmpeg] Merging formats dans la sortie. Ou exécutez ffmpeg -version dans PowerShell.

Optionnel : ajouter FFmpeg au PATH

Pas nécessaire si ffmpeg.exe est à côté de yt-dlp.exe. Mais utile si vous voulez FFmpeg disponible partout.

  1. Recherchez Variables d'environnement dans le menu Démarrer
  2. Ouvrez Path dans les Variables utilisateur
  3. Cliquez sur Nouveau → collez le chemin vers ffmpeg\bin
  4. Cliquez OK, redémarrez le terminal

Dépannage

FFmpeg introuvable ou erreurs de fusion — comment corriger

Erreur "ffmpeg not found" dans yt-dlp
Vérifiez que ffmpeg.exe est exactement dans le même dossier que yt-dlp.exe. Si vous avez ajouté FFmpeg au PATH, fermez toutes les fenêtres du terminal et rouvrez PowerShell — les modifications du PATH ne prennent effet que dans les nouvelles sessions. Exécutez ffmpeg -version pour confirmer qu'il est accessible.
Fusion échouée ou la sortie n'a pas d'audio
Mettez à jour yt-dlp (yt-dlp -U) et FFmpeg vers la dernière version. Les builds FFmpeg obsolètes échouent parfois à fusionner les nouveaux formats de codec. Vérifiez aussi que votre antivirus ne bloque pas ffmpeg.exe.
L'antivirus signale ffmpeg.exe comme une menace
Faux positif. FFmpeg est un outil open source largement utilisé. Téléchargez uniquement depuis gyan.dev ou BtbN (liés depuis la page officielle ffmpeg.org). Ajoutez une exception pour le fichier dans les paramètres antivirus si nécessaire.
Ai-je aussi besoin de ffprobe.exe ?
ffprobe est optionnel. yt-dlp peut l'utiliser pour une détection de format plus précise, mais il n'est pas requis pour les fusions ou conversions de base. Si vous téléchargez le build FFmpeg complet, ffprobe.exe est inclus — placez-le simplement à côté de ffmpeg.exe.
Le MP3 n'a pas d'artwork même avec FFmpeg installé
Assurez-vous d'inclure --embed-thumbnail --embed-metadata dans la commande yt-dlp. FFmpeg doit être présent pour que ces options fonctionnent. Consultez le guide MP3 pour la commande complète.
