Démarrage rapide

Après avoir téléchargé yt-dlp.exe

Quatre étapes du téléchargement à la première vidéo.

Étape 01

Enregistrer dans un dossier

Placez yt-dlp.exe dans un dossier permanent comme C:\Tools\yt-dlp\ — évitez Téléchargements qui peut être nettoyé automatiquement.

Étape 02

Si le navigateur avertit

Cliquez sur Conserver → Conserver quand même dans Edge/Chrome. Si SmartScreen apparaît : Informations complémentaires → Exécuter quand même.

Étape 03

Ajouter FFmpeg (recommandé)

Placez ffmpeg.exe dans le même dossier. Nécessaire pour fusionner la 4K/8K et le MP3. Guide →

Étape 04

Vérifier dans PowerShell

Cliquez sur la barre d'adresse du dossier, tapez powershell, appuyez sur Entrée, puis exécutez yt-dlp --version.

FAQ

Questions fréquentes sur le téléchargement

Ai-je besoin de droits administrateur pour exécuter yt-dlp.exe ?
Non. yt-dlp.exe est portable — aucune installation requise. Les droits administrateur ne sont nécessaires que si vous souhaitez ajouter le dossier au PATH système pour tous les utilisateurs.
Comment convertir en MP3 ?
Exécutez yt-dlp -x --audio-format mp3 --audio-quality 0 "URL" dans PowerShell. FFmpeg doit être dans le même dossier. Consultez le guide YouTube vers MP3.
Comment télécharger une playlist entière ?
Exécutez yt-dlp --yes-playlist "PLAYLIST_URL". Consultez le guide des playlists pour les options de tri, de plages et de reprise.
Comment maintenir yt-dlp à jour ?
Ouvrez PowerShell dans le dossier avec yt-dlp.exe et exécutez yt-dlp -U. Il se remplace automatiquement par la dernière version.
