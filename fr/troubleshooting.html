Dépannage yt-dlp sous Windows — Erreurs courantes et solutions

La plupart des erreurs yt-dlp sous Windows ont des solutions simples. Commencez par mettre à jour — yt-dlp -U — et vérifiez FFmpeg. Cela résout 80 % des problèmes.

Commencez toujours par ici :

Étape 1 — mettre à jour yt-dlp
yt-dlp -U
Étape 2 — vérifier les formats disponibles
yt-dlp -F "URL"

Référence des erreurs

Messages d'erreur yt-dlp et comment les corriger

Erreur "Unable to extract" ou erreur de parsing
Cause : Le site a mis à jour sa structure et l'extracteur de yt-dlp est obsolète.

Solution : Exécutez yt-dlp -U pour obtenir les derniers extracteurs. C'est la solution la plus courante pour cette erreur. Si elle persiste après la mise à jour, le site a peut-être effectué une modification majeure — vérifiez la page des issues GitHub pour les signalements.
Erreur "ffmpeg not found" ou pas d'audio dans le fichier téléchargé
Cause : FFmpeg n'est pas installé ou n'est pas au bon endroit.

Solution : Placez ffmpeg.exe dans le même dossier que yt-dlp.exe. yt-dlp cherche là en premier. Vérifiez avec ffmpeg -version dans PowerShell. Consultez le guide de configuration FFmpeg.
"yt-dlp n'est pas reconnu comme commande"
Cause : PowerShell ne trouve pas yt-dlp.exe car le dossier n'est pas dans le PATH.

Solution — option 1 : Naviguez jusqu'au dossier contenant yt-dlp.exe dans l'Explorateur, cliquez sur la barre d'adresse, tapez powershell et appuyez sur Entrée. Exécutez yt-dlp depuis là.

Solution — option 2 : Ajoutez le dossier yt-dlp au PATH. Consultez le guide d'installation pour les instructions pas à pas.
Erreur HTTP 403 ou "Sign in to confirm your age"
Cause : La vidéo nécessite une authentification ou YouTube limite les requêtes.

Solution : Mettez d'abord à jour yt-dlp : yt-dlp -U. Si la vidéo nécessite une connexion, transmettez les cookies du navigateur : --cookies-from-browser chrome Remplacez chrome par edge ou firefox selon les besoins. Si vous utilisez un VPN, essayez sans — certains nœuds de sortie VPN sont bloqués par YouTube.
Le téléchargement est très lent ou s'arrête continuellement
Cause : Throttling réseau, connexion instable ou limitation de débit YouTube.

Solution : -N 4 --retries 10 --fragment-retries 10 --limit-rate 5M -N 4 active les téléchargements de fragments parallèles. --limit-rate 5M limite la vitesse pour éviter les réinitialisations de connexion. Essayez de télécharger à un autre moment si YouTube limite le débit.
Fusion échouée — vidéo et audio enregistrés en fichiers séparés
Cause : FFmpeg est absent ou obsolète.

Solution : Assurez-vous que ffmpeg.exe est dans le même dossier que yt-dlp.exe et qu'il s'agit d'un build récent. Mettez à jour yt-dlp avec yt-dlp -U. Puis réessayez avec : yt-dlp -f "bestvideo+bestaudio/best" "URL"
Windows SmartScreen bloque yt-dlp.exe
Cause : SmartScreen signale les exécutables portables non signés par défaut — c'est un faux positif.

Solution : Cliquez sur Informations complémentaires → Exécuter quand même. Le fichier est la release officielle non modifiée de GitHub. Vous pouvez aussi faire un clic droit sur yt-dlp.exe → Propriétés → cocher Débloquer en bas → OK.
"Accès refusé" ou l'antivirus bloque le fichier
Cause : Windows ou l'antivirus a bloqué le fichier téléchargé depuis Internet.

Solution : Clic droit sur yt-dlp.exe → Propriétés → cocher Débloquer → OK. Si l'antivirus bloque, ajoutez une exception pour le fichier. Téléchargez toujours depuis la page officielle des releases GitHub.
Échec de vérification du certificat SSL
Cause : Le magasin de certificats Windows est obsolète, ou un proxy/VPN intercepte HTTPS.

Solution : Mettez à jour Windows (Paramètres → Windows Update), puis réessayez. Si vous êtes derrière un proxy d'entreprise, contactez votre équipe IT. En dernier recours uniquement : --no-check-certificate — non recommandé à long terme car désactive la validation des certificats.
Erreur "URL non supportée"
Cause : Le site n'est pas supporté par yt-dlp, ou le format d'URL a changé.

Solution : Mettez à jour yt-dlp avec yt-dlp -U. Vérifiez la liste des sites supportés. Assurez-vous d'utiliser la bonne URL vidéo — les URLs de playlist et de chaîne ont des formats différents.
