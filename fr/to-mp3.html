YouTube en MP3 avec yt-dlp sous Windows — Meilleure qualité

Téléchargez l'audio YouTube en MP3 avec pochette et tags intégrés. Fonctionne pour les vidéos individuelles, les playlists et les chaînes. Nécessite FFmpeg pour la conversion.

⬇ Télécharger yt-dlp.exe Configurer FFmpeg →

Commande recommandée

yt-dlp YouTube en MP3 — commande de meilleure qualité

Cette seule commande gère tout : convertit en MP3, intègre la pochette et les tags ID3.

MP3 de meilleure qualité avec artwork et tags (recommandé)
yt-dlp -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --embed-metadata "VIDEO_URL"

Ce que fait chaque option

  • -x — extraire l'audio uniquement
  • --audio-format mp3 — convertir en MP3
  • --audio-quality 0 — qualité VBR maximale
  • --embed-thumbnail — intégrer la pochette
  • --embed-metadata — intégrer les tags ID3

Prérequis

  • ffmpeg.exe doit être dans le même dossier que yt-dlp.exe
  • Sans FFmpeg : la conversion et l'intégration d'artwork ne fonctionneront pas
  • Consultez le guide de configuration FFmpeg si nécessaire
Note sur la qualité : YouTube diffuse l'audio au format Opus à ~128–160 kbps. Convertir en "MP3 320 kbps" cible ce débit mais ne peut pas ajouter de qualité au-delà de la source originale. Utilisez --audio-quality 0 pour la meilleure sortie possible à partir de ce qui est disponible.

Plus d'options

Toutes les commandes de téléchargement MP3 yt-dlp

Copiez-collez dans PowerShell. Remplacez l'URL par votre lien vidéo ou playlist.

Débit constant — cible ~192 kbps
yt-dlp -x --audio-format mp3 --audio-quality 192K --embed-thumbnail --embed-metadata "VIDEO_URL"
Sans ré-encodage — conserver l'audio Opus original (meilleure qualité réelle)
yt-dlp -f ba -x --audio-format best "VIDEO_URL"
Convertir une playlist YouTube entière en MP3
yt-dlp -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --embed-metadata --yes-playlist -o "%(playlist_index)02d-%(title)s.%(ext)s" "PLAYLIST_URL"
Traiter plusieurs URLs depuis un fichier texte
yt-dlp -a links.txt -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --embed-metadata
Enregistrer dans un dossier personnalisé avec nommage artiste/titre
yt-dlp -x --audio-format mp3 --audio-quality 0 -o "C:/Musique/%(artist,channel)s/%(title)s.%(ext)s" "VIDEO_URL"

Conseils

Conseils pour les noms de fichiers, tags et artwork

Templates de sortie

  • %(title)s — titre de la vidéo
  • %(artist)s — artiste (si disponible)
  • %(channel)s — nom de la chaîne
  • %(playlist_index)02d — numéro de piste avec zéros
  • %(ext)s — extension du fichier (mp3)

Ignorer les déjà téléchargés

Ajoutez ceci à n'importe quelle commande pour ignorer les fichiers déjà convertis — utile pour les grandes playlists :

--download-archive downloaded.txt

Crée un fichier journal et ignore les entrées lors des réexécutions.

FAQ

YouTube en MP3 avec yt-dlp — questions fréquentes

Un MP3 de 320 kbps de YouTube est-il vraiment en haute qualité ?
Le fichier MP3 sera encodé à 320 kbps, mais la qualité audio réelle dépend de la source. YouTube diffuse l'audio Opus à ~128–160 kbps. Ré-encoder en MP3 320 kbps ne peut pas ajouter des détails absents de l'original. Si la qualité audio est votre priorité, utilisez -f ba -x --audio-format best pour conserver le flux Opus original sans ré-encodage.
Pourquoi mon MP3 n'a-t-il pas de pochette ni de tags ?
FFmpeg doit être dans le même dossier que yt-dlp.exe, et vous avez besoin de --embed-thumbnail --embed-metadata dans la commande. Sans FFmpeg, yt-dlp ne peut pas écrire la pochette ni les tags ID3. Consultez le guide de configuration FFmpeg.
Comment convertir une playlist YouTube entière en MP3 ?
Passez une URL de playlist directement — yt-dlp traitera toutes les vidéos. Ajoutez --yes-playlist si yt-dlp demande confirmation, et utilisez -o "%(playlist_index)02d-%(title)s.%(ext)s" pour des noms de fichiers numérotés et triés.
Où sont enregistrés les fichiers MP3 ?
Par défaut, dans le dossier où vous avez lancé PowerShell. Utilisez -o "C:/Musique/%(title)s.%(ext)s" pour contrôler le chemin de sortie. Les barres obliques normales fonctionnent sous Windows.
Puis-je convertir plusieurs URLs en une seule fois ?
Oui. Créez un fichier texte brut (ex. : links.txt) avec une URL par ligne, puis exécutez yt-dlp -a links.txt -x --audio-format mp3 --audio-quality 0.
