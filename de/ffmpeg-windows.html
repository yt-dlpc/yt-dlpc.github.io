FFmpeg für Windows — Setup für yt-dlp

4K/8K‑Video+Audio‑Merges und MP3‑Konvertierungen aktivieren, indem du ffmpeg.exe zu deinem Windows‑Setup hinzufügst. Am schnellsten und einfachsten: Lege ffmpeg.exe neben yt-dlp.exe.

⬇️ FFmpeg für Windows herunterladen yt-dlp.exe holen

Hinweis: Windows‑Builds werden von vertrauenswürdigen Community‑Maintainer:innen bereitgestellt, die auf der offiziellen FFmpeg‑Website gelistet sind. Wähle einen aktuellen Build und entpacke ihn lokal.
• Funktioniert auf Windows 11/10

Schnellsetup (empfohlen)

  1. Lade einen Windows‑Build über die offiziellen FFmpeg‑Downloads herunter.
  2. Entpacke das Archiv und finde ffmpeg.exe (meist im Ordner bin).
  3. Kopiere ffmpeg.exe in denselben Ordner wie yt-dlp.exe.
  4. Teste ein 4K‑Video; du solltest [ffmpeg] Merging formats in der Ausgabe sehen.

Prüfen, ob FFmpeg erkannt wird

Führe in PowerShell einen kurzen Download aus:

yt-dlp -f "bestvideo+bestaudio/best" "https://www.youtube.com/watch?v=VIDEO_ID"

Achte auf [ffmpeg] Merging formats. Alternativ prüfen mit:

ffmpeg -version

(Optional) FFmpeg zum PATH hinzufügen

  • GUI: Umgebungsvariablen → Benutzervariablen → Path → Neu → FFmpeg‑bin‑Ordner einfügen → OK.
  • PowerShell (aktueller Benutzer): setx PATH "%PATH%;C:\\Path\\to\\ffmpeg\\bin" (Terminal neu starten).

Für yt-dlp ist PATH optional, wenn ffmpeg.exe neben der ausführbaren Datei liegt.

Best Practices

  • Halte FFmpeg auf dem neuesten stabilen Stand für Codec‑Fixes und Performance.
  • Lege ffprobe.exe neben ffmpeg.exe für erweiterte Formaterkennung (optional).
  • Bewahre yt-dlp.exe und ffmpeg.exe in einem dedizierten Ordner wie C:\\Tools\\yt-dlp auf.

Häufige Fallstricke

  • „ffmpeg not found“: Stelle sicher, dass ffmpeg.exe im selben Ordner wie yt-dlp.exe liegt oder zum PATH hinzugefügt wurde.
  • Merge‑Fehler: Aktualisiere FFmpeg und yt-dlp; prüfe, ob Antivirus nicht blockiert; ggf. neu herunterladen.
  • Kein Artwork/Tags in MP3: Nutze --embed-thumbnail --embed-metadata und stelle sicher, dass FFmpeg vorhanden ist.

FAQ

Wird FFmpeg für jeden Download benötigt?

Nein. Es wird für das Zusammenführen getrennter Video/Audio‑Streams (typisch bei 4K/8K) und für Audiokonvertierungen (z. B. MP3) benötigt. Für einfache Single‑Stream‑Downloads kann yt-dlp auch ohne FFmpeg funktionieren.

Wo sollte ich ffprobe.exe ablegen?

Am besten im selben Ordner wie ffmpeg.exe. yt-dlp kann ffprobe für eine bessere Format‑Erkennung nutzen, es ist aber optional.

Kann ich statt Kopieren den PATH verwenden?

Ja. Füge den FFmpeg‑bin‑Ordner zum PATH hinzu; yt-dlp findet es dann systemweit.
