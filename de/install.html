Schritt-für-Schritt-Installation

yt-dlp in 6 Schritten unter Windows installieren

Funktioniert auf Windows 11 und Windows 10 (64-Bit). Kein Installationsassistent, keine Adminrechte erforderlich.

Schritt 01

yt-dlp.exe herunterladen

Zur Download-Seite gehen und yt-dlp.exe vom offiziellen GitHub-Release speichern.

Schritt 02

In einem dauerhaften Ordner speichern

Die Datei an einen festen Ort verschieben, z. B. C:\Tools\yt-dlp\. Downloads meiden — Dateien dort werden manchmal gelöscht.

Schritt 03

PowerShell in diesem Ordner öffnen

Im Explorer die Adressleiste anklicken, powershell eingeben und Enter drücken. PowerShell öffnet sich direkt im richtigen Ordner.

Schritt 04

Funktion prüfen

yt-dlp --version ausführen. Es sollte eine Versionsnummer wie 2025.xx.xx erscheinen — dann funktioniert alles.

Schritt 05

Auf die neueste Version aktualisieren

yt-dlp -U ausführen. Es lädt sich herunter und ersetzt sich durch das neueste Release. Regelmäßig wiederholen.

Schritt 06

FFmpeg installieren (empfohlen)

ffmpeg.exe herunterladen und in denselben Ordner legen. Für 4K/8K-Zusammenführungen und MP3 erforderlich. Vollständige Anleitung →

Optional, aber empfohlen

yt-dlp zum PATH unter Windows hinzufügen

Mit yt-dlp im PATH kann yt-dlp aus jedem Ordner in PowerShell oder CMD gestartet werden — ohne jedes Mal in den yt-dlp-Ordner navigieren zu müssen.

GUI-Methode (empfohlen)

  1. Win drücken und Umgebungsvariablen suchen
  2. Systemumgebungsvariablen bearbeiten klicken
  3. Unter Benutzervariablen Pfad → Bearbeiten wählen
  4. Neu klicken und Ordnerpfad einfügen, z. B. C:\Tools\yt-dlp
  5. Alle Dialoge mit OK bestätigen, dann Terminal neu starten

PowerShell-Methode

Einmal in PowerShell ausführen — setzt PATH nur für den eigenen Benutzer, keine Adminrechte nötig:

setx PATH "%PATH%;C:\Tools\yt-dlp"

PowerShell nach diesem Befehl schließen und neu öffnen. Danach mit yt-dlp --version aus einem beliebigen Verzeichnis testen.

Hinweis: Falls nach dem Hinzufügen zum PATH „yt-dlp wird nicht erkannt" erscheint, alle Terminal-Fenster schließen und neu öffnen — PATH-Änderungen gelten nur für neue Sitzungen.

Dringend empfohlen

FFmpeg für yt-dlp unter Windows einrichten

FFmpeg ist ein kostenloses Tool, das yt-dlp zum Zusammenführen von Video- und Audiostreams verwendet. Ohne FFmpeg funktionieren 4K/8K-Downloads und MP3-Konvertierung nicht.

Was FFmpeg ermöglicht

  • 4K- und 8K-Video-Downloads (separate Streams werden zusammengeführt)
  • MP3-, AAC-, FLAC-, Opus-Audio-Extraktion
  • HDR-Videoformat-Verarbeitung
  • Untertitel in MKV-Dateien einbetten

Schnelle Einrichtung

  1. ffmpeg.exe aus der FFmpeg-Anleitung herunterladen
  2. ffmpeg.exe in denselben Ordner wie yt-dlp.exe legen
  3. yt-dlp erkennt es automatisch — keine Konfiguration nötig

Häufige Probleme

Fehlerbehebung bei der yt-dlp-Installation unter Windows

Windows SmartScreen-Warnung beim Ausführen von yt-dlp.exe
Weitere Informationen → Trotzdem ausführen klicken. SmartScreen markiert neue ausführbare Dateien standardmäßig. yt-dlp.exe ist das unveränderte offizielle Open-Source-Release von GitHub.
„yt-dlp wird nicht erkannt" in PowerShell
Zwei Möglichkeiten: yt-dlp direkt aus seinem Ordner ausführen oder den Ordner zum PATH hinzufügen (siehe Abschnitt oben). Nach dem Hinzufügen zum PATH alle Terminal-Fenster schließen und neu öffnen.
„Zugriff verweigert" oder Datei blockiert
Rechtsklick auf yt-dlp.exe → Eigenschaften → unten Entsperren ankreuzen, falls das Kontrollkästchen erscheint → OK. Dadurch wird die Windows-Einschränkung „aus dem Internet heruntergeladen" entfernt.
4K/8K-Download liefert nur Video oder nur Audio
FFmpeg ist nicht installiert oder wird nicht gefunden. ffmpeg.exe in denselben Ordner wie yt-dlp.exe legen. yt-dlp erkennt es automatisch und führt die Streams zusammen.
yt-dlp -U meldet „already up to date" aber ich nutze eine alte Version
PowerShell als Administrator ausführen und erneut versuchen. Oder die neueste yt-dlp.exe manuell von der Download-Seite herunterladen und die alte Datei ersetzen.
