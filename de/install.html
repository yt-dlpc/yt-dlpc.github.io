yt-dlp unter Windows installieren

Kurzanleitung zur Einrichtung von yt-dlp.exe unter Windows 11/10, Hinzufügen zum PATH, Prüfen, Aktualisieren und Aktivieren von FFmpeg für 4K/8K und MP3.

⬇️ yt-dlp.exe herunterladen

Tipp: Du kannst yt-dlp.exe ohne Installation ausführen. Das Hinzufügen zum PATH macht es nur aus jedem Ordner verfügbar.
• Funktioniert auf Windows 11/10

Schritt‑für‑Schritt

  1. Lade yt-dlp.exe vom offiziellen GitHub‑Release: neuester Build.
  2. Verschiebe sie in einen Ordner wie C:\\Tools\\yt-dlp oder C:\\Users\\YOU\\Downloads.
  3. (Optional) Füge den Ordner deinem PATH hinzu:
    • GUI‑Methode: Start → nach „Umgebungsvariablen“ suchen → Umgebungsvariablen bearbeiten → unter Benutzervariablen Path öffnen → Neu → deinen Ordner einfügen (z. B. C:\\Tools\\yt-dlp) → OK.
    • PowerShell (aktueller Benutzer): setx PATH "%PATH%;C:\\Tools\\yt-dlp" (danach Terminal neu starten).
  4. Öffne PowerShell im Ordner (Explorer → Adressleiste → powershell tippen → Enter) und führe aus:
    yt-dlp --version
  5. Halte es aktuell:
    yt-dlp -U

FFmpeg (empfohlen)

FFmpeg ermöglicht das Zusammenführen getrennter Video/Audio‑Streams (4K/8K) und Audiokonvertierungen (z. B. MP3).

  1. Lade ffmpeg.exe aus offiziellen Quellen herunter.
  2. Lege ffmpeg.exe neben yt-dlp.exe in denselben Ordner.
  3. Prüfe durch einen Download in hoher Qualität; yt-dlp sollte FFmpeg automatisch erkennen und Formate mergen.

Alle Details: FFmpeg unter Windows einrichten.

Nach der Installation: Checks

Kurzer Test in PowerShell:

yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"

Version und Update‑Status prüfen:

yt-dlp --version yt-dlp -U

Aus jedem Ordner starten

Sobald der Ordner im PATH ist, kannst du yt-dlp überall ausführen. Ansonsten aus seinem Ordner starten oder den vollständigen Pfad angeben:

C:\\Tools\\yt-dlp\\yt-dlp.exe --version

Fehlerbehebung

  • SmartScreen‑Warnung? Klicke Weitere Informationen → Trotzdem ausführen. Üblich bei portablen Open‑Source‑Tools.
  • „Befehl nicht gefunden“ nach PATH‑Update? Terminal schließen und neu öffnen (PATH wird beim Sitzungsstart geladen).
  • „Zugriff verweigert“ oder blockiert? Rechtsklick auf yt-dlp.exe → Eigenschaften → Zulassen/Entsperren (falls vorhanden) aktivieren → OK.
  • Merge‑Fehler bei 4K/8K? Stelle sicher, dass ffmpeg.exe im selben Ordner wie yt-dlp.exe liegt.

Siehe auch: Befehls‑Cheatsheet • Ausführliche Fehlerbehebung

FAQ

Brauche ich Admin‑Rechte für die Installation?

Nein. yt-dlp.exe ist portabel. Admin ist nur nötig, wenn du PATH für alle Benutzer setzen willst; für deinen Benutzer nutze die GUI‑Methode oder setx.

Funktioniert das auf Windows 7/8 oder ARM?

yt-dlp zielt primär auf moderne Windows‑Versionen. Ältere Systeme können eine Python‑basierte Einrichtung benötigen. Für ARM64 prüfe die offiziellen Release‑Hinweise zur Kompatibilität.

Wie aktualisiere ich auf die neueste Version?

Führe yt-dlp -U aus. Dadurch wird die ausführbare Datei direkt aktualisiert.
