Instalación paso a paso

Cómo instalar yt-dlp en Windows en 6 pasos

Compatible con Windows 11 y Windows 10 (64 bits). Sin asistente de instalación, sin permisos de administrador.

Paso 01

Descargar yt-dlp.exe

Ve a la página de descarga y guarda yt-dlp.exe desde el release oficial de GitHub.

Paso 02

Guardar en una carpeta dedicada

Mueve el archivo a una ubicación permanente, p. ej.: C:\Tools\yt-dlp\. Evita Descargas — los archivos allí pueden limpiarse automáticamente.

Paso 03

Abrir PowerShell en esa carpeta

En el Explorador, haz clic en la barra de direcciones, escribe powershell y pulsa Enter. PowerShell se abre directamente en la carpeta correcta.

Paso 04

Verificar que funciona

Ejecuta yt-dlp --version. Debe aparecer un número de versión como 2025.xx.xx. Si es así, funciona correctamente.

Paso 05

Actualizar a la última versión

Ejecuta yt-dlp -U. Descarga y se sustituye por el release más reciente. Hazlo regularmente.

Paso 06

Instalar FFmpeg (recomendado)

Descarga ffmpeg.exe y colócalo en la misma carpeta. Necesario para fusionar 4K/8K y MP3. Guía completa →

Opcional pero recomendado

Cómo añadir yt-dlp al PATH en Windows

Añadir yt-dlp al PATH permite ejecutar yt-dlp desde cualquier carpeta en PowerShell o CMD — sin necesidad de navegar hasta la carpeta de yt-dlp cada vez.

Método gráfico (recomendado)

  1. Pulsa Win y busca Variables de entorno
  2. Haz clic en Editar las variables de entorno del sistema
  3. En Variables de usuario, selecciona Path → Editar
  4. Haz clic en Nuevo y pega la ruta de la carpeta, p. ej.: C:\Tools\yt-dlp
  5. Haz clic en Aceptar en todos los diálogos y reinicia el terminal

Método PowerShell

Ejecuta esto una vez en PowerShell — define el PATH solo para tu usuario, sin permisos de administrador:

setx PATH "%PATH%;C:\Tools\yt-dlp"

Cierra y vuelve a abrir PowerShell después. Prueba con yt-dlp --version desde cualquier directorio.

Nota: Si aparece "yt-dlp no se reconoce como comando" después de añadir al PATH, cierra todas las ventanas del terminal y vuelve a abrirlas — los cambios del PATH solo tienen efecto en nuevas sesiones.

Verificaciones post-instalación

Verificar la instalación y mantener yt-dlp actualizado

Verificar la versión instalada
yt-dlp --version
Actualizar al último release
yt-dlp -U
Probar con una descarga real (mejor calidad)
yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"
Si yt-dlp no está en el PATH — ejecuta directamente desde la carpeta
C:\Tools\yt-dlp\yt-dlp.exe --version

Problemas comunes

Solucionar problemas de instalación de yt-dlp en Windows

Advertencia de SmartScreen de Windows al ejecutar yt-dlp.exe
Haz clic en Más información → Ejecutar de todos modos. SmartScreen señala nuevos ejecutables por defecto. yt-dlp.exe es el release oficial open source de GitHub sin modificaciones.
"yt-dlp no se reconoce" en PowerShell
Dos opciones: ejecuta yt-dlp directamente desde su carpeta, o añade la carpeta al PATH (ver sección anterior). Después de añadir al PATH, cierra todas las ventanas del terminal y vuelve a abrirlas.
"Acceso denegado" o archivo bloqueado
Haz clic derecho en yt-dlp.exe → Propiedades → en la parte inferior marca Desbloquear si aparece la casilla → haz clic en Aceptar. Esto elimina la restricción de Windows "descargado de internet".
La descarga 4K/8K devuelve solo vídeo o solo audio
FFmpeg no está instalado o no se encuentra. Coloca ffmpeg.exe en la misma carpeta que yt-dlp.exe. yt-dlp lo detectará automáticamente y fusionará los flujos.
