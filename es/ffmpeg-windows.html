Guía de configuración

Cómo instalar FFmpeg para yt-dlp en Windows

Cuatro pasos, lleva menos de 4 minutos.

Paso 01

Descargar FFmpeg para Windows

Ve a ffmpeg.org/download.html y elige un build Windows. Fuentes recomendadas: gyan.dev o BtbN — ambas listadas en la página oficial.

Paso 02

Extraer el archivo

Extrae el archivo .zip o .7z. Abre la carpeta extraída y navega hasta la subcarpeta bin — allí encontrarás ffmpeg.exe.

Paso 03

Colocar junto a yt-dlp.exe

Copia ffmpeg.exe en la misma carpeta que yt-dlp.exe (p. ej.: C:\Tools\yt-dlp\). yt-dlp lo detecta automáticamente — sin configuración adicional.

Paso 04

Verificar que funciona

Descarga cualquier vídeo con yt-dlp. Deberías ver [ffmpeg] Merging formats en la salida. O ejecuta ffmpeg -version en PowerShell.

Verificar & PATH opcional

Verificar que FFmpeg funciona con yt-dlp

Verificar en PowerShell

Verificar la versión de FFmpeg
ffmpeg -version
Probar la fusión con una descarga real
yt-dlp -f "bestvideo+bestaudio/best" "URL"

Busca [ffmpeg] Merging formats into en la salida.

Opcional: añadir FFmpeg al PATH

No es necesario si ffmpeg.exe está junto a yt-dlp.exe. Pero útil si quieres FFmpeg disponible en cualquier lugar.

  1. Busca Variables de entorno en el menú Inicio
  2. Abre Path en Variables de usuario
  3. Haz clic en Nuevo → pega la ruta a ffmpeg\bin
  4. Haz clic en Aceptar, reinicia el terminal

Solución de problemas

FFmpeg no encontrado o errores de fusión — cómo solucionarlo

Error "ffmpeg not found" en yt-dlp
Comprueba que ffmpeg.exe está exactamente en la misma carpeta que yt-dlp.exe. Si añadiste FFmpeg al PATH, cierra todas las ventanas del terminal y vuelve a abrir PowerShell — los cambios en el PATH solo tienen efecto en nuevas sesiones. Ejecuta ffmpeg -version para confirmar que es accesible.
Fusión fallida o la salida no tiene audio
Actualiza yt-dlp (yt-dlp -U) y FFmpeg a la última versión. Los builds desactualizados de FFmpeg a veces fallan al fusionar nuevos formatos de códec. Comprueba también que el antivirus no está bloqueando ffmpeg.exe.
El antivirus señala ffmpeg.exe como amenaza
Falso positivo. FFmpeg es una herramienta open source ampliamente usada. Descarga solo desde gyan.dev o BtbN (enlazados desde la página oficial ffmpeg.org). Añade una excepción para el archivo en la configuración del antivirus si es necesario.
¿También necesito ffprobe.exe?
ffprobe es opcional. yt-dlp puede usarlo para una detección de formato más precisa, pero no es necesario para las fusiones o conversiones básicas. Si descargas el build completo de FFmpeg, ffprobe.exe está incluido — solo colócalo junto a ffmpeg.exe.
El MP3 no tiene carátula aunque FFmpeg está instalado
Asegúrate de incluir --embed-thumbnail --embed-metadata en el comando de yt-dlp. FFmpeg debe estar presente para que estos flags funcionen. Consulta la guía MP3 para el comando completo.
