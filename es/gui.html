Guía de configuración

Cómo configurar una GUI de yt-dlp en Windows

La mayoría de las GUI siguen el mismo proceso de configuración. En menos de 5 minutos.

Paso 01

Descargar yt-dlp.exe

Obtén el último yt-dlp.exe en nuestra página de descarga. Guárdalo en C:\Tools\yt-dlp\

Paso 02

Descargar la GUI

Elige una GUI de la lista anterior. Descarga el release desde la página GitHub del proyecto — busca un .exe o .zip en la sección Releases.

Paso 03

Apuntar la GUI hacia yt-dlp.exe

La mayoría de las GUI piden la ruta hacia yt-dlp.exe al primer uso. Navega hasta C:\Tools\yt-dlp\ y selecciona el archivo.

Paso 04

Añadir FFmpeg

Descarga ffmpeg.exe y colócalo junto a yt-dlp.exe. Necesario para 4K/8K y MP3. Guía →

Paso 05

Definir carpeta de salida

En la configuración de la GUI, elige dónde se guardarán las descargas. Una carpeta dedicada como C:\Vídeos\ mantiene todo organizado.

Paso 06

Pegar URL y descargar

Pega una URL de YouTube en la GUI, elige formato y calidad, y haz clic en Descargar. Listo.

FAQ

GUI yt-dlp para Windows — preguntas frecuentes

¿Existe una GUI oficial para yt-dlp?
No. yt-dlp es una herramienta de línea de comandos y los desarrolladores no han publicado una GUI oficial. Todas las interfaces listadas aquí son proyectos de la comunidad.
¿Todavía necesito instalar yt-dlp al usar una GUI?
Depende de la GUI. Algunas incluyen yt-dlp (como Parabolic), mientras que otras piden que apuntes a tu propio yt-dlp.exe. Consulta la documentación de la GUI — si incluye yt-dlp, asegúrate de que esté actualizado.
¿Necesito FFmpeg al usar una GUI de yt-dlp?
Sí, para descargas 4K/8K y conversión a MP3. Algunas GUI incluyen FFmpeg; otras necesitan que configures la ruta. Coloca ffmpeg.exe en la misma carpeta que yt-dlp.exe y la mayoría de las GUI lo detectarán automáticamente. Consulta la guía de configuración de FFmpeg.
La GUI dice "yt-dlp no encontrado" — ¿qué hacer?
Ve a la Configuración de la GUI y define manualmente la ruta de yt-dlp. Navega hasta la carpeta donde guardaste yt-dlp.exe (p. ej.: C:\Tools\yt-dlp\yt-dlp.exe) y selecciona el archivo.
¿Prefieres la línea de comandos al final?
Entendido — el CLI da control total. Consulta el cheatsheet de comandos yt-dlp para comandos de copiar y pegar que cubren todos los casos de uso comunes.
