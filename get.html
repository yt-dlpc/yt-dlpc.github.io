How to handle the ISO

An ISO is a disk image. Here are safe ways to open it in Windows.

  1. Double‑click the ISO file — Windows will mount it as a virtual drive. Open the drive and run setup.exe if present.
  2. Right‑click → Mount — same as double‑click.
  3. Need files inside? Use 7‑Zip (official archiver) to extract without mounting.
  4. PowerShell (advanced): run as admin: Mount-DiskImage -ImagePath "C:\\path\\to\\file.iso"
  5. After mounting, find the drive in File Explorer, open it and start the installer. If SmartScreen appears, see tips below.
